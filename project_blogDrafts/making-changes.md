### Making Changes

So, in the past week I experimented a bit with making changes to the CSS modules used to provide a default layout.

I've also been making a list of features to include in the site to make it more interactive- and to also get some practise with new _functionality_. When it comes to more dynamic, interactive elements- I can quickly get out of my comfort zone trying to pin down where to start to introduce something like a search bar, or a form input element.

Before, this would normally mean that I would then put it off... because it's scary. Not this time though- _no_. This time, I'll be making my plan, looking at examples, and then implementing new features at a nice, _slow_ pace to help me understand what the code is _actually_ doing... or _needs_ to do.
